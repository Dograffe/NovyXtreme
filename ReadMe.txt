General Plugin Info:

Permission nodes:
  novyXtreme.nxgo : Allows players to use /nxgo
  novyXreme.nxlistown : Gives player a list of stargates owned by them when using /nxlist
  novyXtreme.nxlistall : Gives player a list of all stargates when using /nxlist
  novyXtreme.nxremoveany : Player can remove any stargate with /nxremove
  novyXtreme.nxremoveown : Player can remove stargates owned by them with /nxremove
